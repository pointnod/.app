# Version Android Standalone (APK)

Cette version utilise **Capacitor** pour encapsuler le code web.

## Compilation
1. Faire le build du dossier `web-mobile`.
2. Copier le résultat dans le dossier `www` de cette plateforme.
3. Exécuter `npx cap sync android`.
4. Ouvrir via Android Studio Portable pour générer l'APK.
